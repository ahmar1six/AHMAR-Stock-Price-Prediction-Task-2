# Stock Price Prediction Using Machine Learning

## Task Objective
The objective of this task is to predict the next day's closing price of a stock using historical stock data and machine learning techniques.

## Dataset Used
Historical stock data of Amazon (AMZN) was collected using the yfinance Python library from Yahoo Finance.

## Features Used
- Open
- High
- Low
- Volume

## Target Variable
Next Day Closing Price

## Model Applied
Linear Regression

## Tools and Libraries
- Python
- Pandas
- Matplotlib
- Scikit-learn
- yfinance

## Key Results
The model was able to learn the relationship between historical stock features and closing price. The predicted values were compared with actual values using visualization and Mean Squared Error.
